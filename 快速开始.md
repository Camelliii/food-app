# 快速开始指南

## 1. 安装依赖

在 `food-app` 目录下运行：

```bash
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

应用将在浏览器中自动打开，通常地址为 `http://localhost:5173`

## 3. 开始使用

应用已经包含了示例数据，你可以：

1. **浏览菜谱库** - 查看示例菜谱，尝试搜索和分类筛选
2. **添加菜谱到菜单** - 点击菜谱详情页的"加入菜单"按钮
3. **查看采购清单** - 在待办清单页面自动生成需要购买的食材
4. **管理库存** - 在冰箱库存页面添加、编辑、删除食材
5. **完成烹饪流程** - 标记菜谱完成，点击"开饭"消耗库存

## 功能亮点

- ✅ 智能库存检查 - 自动显示哪些菜谱可以制作
- ✅ 自动生成采购清单 - 根据菜单和库存自动计算需要购买的食材
- ✅ 一键入库 - 采购完成后批量添加到库存
- ✅ 库存消耗确认 - 烹饪完成后智能扣减库存
- ✅ 常备物品管理 - 标记常备物品，不参与库存扣减
- ✅ 数据持久化 - 所有数据保存在浏览器本地

## 提示

- 所有数据保存在浏览器的 LocalStorage 中
- 清除浏览器数据会删除所有内容
- 建议定期备份重要数据（可以导出 LocalStorage 数据）
